Remove WP Block Editor disables the new wp editor for posts, pages and even custom post types.

== Description ==

Recently wordpress introduced a new page and posts block editor whose look and feel is totally different then the classic wordpress page editor. We all love the classic editor and some of you might not be ready to accept this change as you feel there is a risk of data being changed or lost as you are less familiar with the new one. So if you are using wordpress latest version and want to revert to the classic wordpress editor then this should be the best plugin to disable the new editor.

== Installation ==

Below are the steps to install New Page Link plugin

1. Upload plugin to the `/wp-content/plugins/` directory
2. Go to the plugins settings page and select the checkboxes as per your need.
3. Open any page or post in edit mode and if you have checked checkboxes in previous step the classic editor will be back for you.

== Frequently Asked Questions ==
=How is this plugin different than other plugins ?
It gives you more control as you also get an options page to disable wp new block editor at selected places like custom post types for instance.
